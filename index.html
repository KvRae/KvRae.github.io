Academic Projects

Here is some of the projects that i made during my learning journey

BackAppX


BackAppX is a MERN stack BaaS application that allows users to create and manage their own backends without needing to write any server-side code.

Easy Kitchen


User-friendly and informative food mobile app that will provide recipes and tips on healthy food. It adopts graphic elements and correspondent color solutions to showcase the listed healthy dishes or recipes in an intuitive and visual way.

TT-Docs


Web application manages and exports all of the papers related to employees, allowing for one-click management and exportation instead of manual processing. The application is set up on a local server most features are unavailable on the Github repository.

Tunesports


The project was created with my classmates at esprit and is available on the web, desktop, and mobile. Tunesports is an e-sports news application that covers everything about electronic video games, selling products, coaching, and much more.

Beads Leads


Beads Leads is a strategy turn-based game inspired by the three men's Morris game's style and pretty similar to tic tac toe, the goal is to align your beads in a row to win the game. Available on both web and android versions.

Invade Space


In the past, the first game that I created with the game engine "Godot" was called Invade Space. This 2D space shooter game will transport you to space.
